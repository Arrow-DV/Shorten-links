# Shorten Links
> This Tool Short Links Using Python ( Gui )

 ## installation
    - Clone this repo `https://github.com/Arrow-DV/Shorten-links/`
    - Extract The Zip File
    - cd ShortenLinks
    - Install the requirements `pip install -r requirements.txt`
    - python main.py
